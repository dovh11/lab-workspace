# Group Expense Manager (GEM) - Web Application

## 📝 Giới thiệu đề tài
Hệ thống Quản lý và Chia sẻ Chi phí Nhóm là ứng dụng Web giúp các nhóm (sinh viên, bạn bè, đồng nghiệp) ghi chép hóa đơn và tự động tính toán quyết toán nợ một cách minh bạch. Đề tài được xây dựng trong khuôn khổ học phần **Kỹ thuật phần mềm (CSE702025)** tại Đại học Phenikaa.

## 🎯 Mục tiêu học thuật
Dự án áp dụng đầy đủ quy trình và chuẩn đầu ra của học phần:
- **SDLC:** Áp dụng vòng đời phát triển phần mềm chuẩn (Phân tích - Thiết kế - Lập trình - Kiểm thử).
- **OOAD:** Phân tích và thiết kế hướng đối tượng thông qua các biểu đồ UML (Use Case, Class, Sequence Diagram).
- **Architecture:** Triển khai theo mô hình kiến trúc phần mềm hiện đại (MVC/Client-Server).
- **Quality Assurance:** Xây dựng các kịch bản kiểm thử đảm bảo độ chính xác của thuật toán quyết toán.

## 🚀 Tính năng cốt lõi
1. **Quản lý nhóm:** Tạo nhóm và mời thành viên tham gia.
2. **Ghi chép chi tiêu:** Nhập hóa đơn, mô tả và phân loại chi phí.
3. **Quyết toán tự động:** Thuật toán tối ưu hóa số lượng giao dịch giữa các thành viên.
4. **Báo cáo:** Thống kê chi tiêu trực quan theo thời gian và hạng mục.

## 🛠 Công nghệ sử dụng
- **Backend:** Python (Flask/Django) - Phù hợp với nội dung đào tạo Module 3.
- **Frontend:** HTML5, CSS3, JavaScript.
- **Database:** SQLite/PostgreSQL.
- **Version Control:** Git & GitHub.
